# Daily Todo List - 安装指南

## 📱 应用简介

这是一个简洁的每日任务清单管理工具，支持以下功能：

- ✅ 按日期流式展示任务
- ✅ 设置任务优先级和截止日期
- ✅ 过期前一天红色提醒
- ✅ 自动隐藏已完成的历史日期
- ✅ 数据本地存储，支持离线使用

---

## 🚀 安装方式

### 方式一：PWA 在线安装（推荐）

**特点**：无需下载，直接安装到设备，自动更新

#### Windows 安装步骤：

1. 使用 **Microsoft Edge** 或 **Google Chrome** 浏览器打开应用网址
2. 地址栏右侧会出现 **"安装"** 或 **"应用可用"** 图标
3. 点击图标，选择 **"安装"**
4. 应用将出现在开始菜单和任务栏

#### Android 安装步骤：

1. 使用 **Chrome** 或 **Edge** 浏览器打开应用网址
2. 浏览器会弹出 **"添加到主屏幕"** 提示
3. 点击 **"添加"**，或手动点击菜单 → "添加到主屏幕"
4. 应用图标将出现在主屏幕

#### iOS 安装步骤：

1. 使用 **Safari** 浏览器打开应用网址
2. 点击底部中间的 **分享按钮**
3. 向下滚动，选择 **"添加到主屏幕"**
4. 点击右上角的 **"添加"**

---

### 方式二：使用在线打包工具

**特点**：生成原生安装包（APK/MSIX），可离线分发

#### 步骤 1：部署 PWA 文件

将 `dist` 文件夹部署到任意静态网站托管服务：

- **免费选项**：
  - GitHub Pages
  - Vercel
  - Netlify
  - 腾讯云静态网站托管
  - 阿里云 OSS

- **部署后获得网址**：`https://your-domain.com/`

#### 步骤 2：使用 PWABuilder 打包

1. 访问 [https://www.pwabuilder.com](https://www.pwabuilder.com)
2. 输入你的 PWA 网址
3. 点击 **"Start"**
4. 选择要生成的包类型：
   - **Android**：生成 APK 文件
   - **Windows**：生成 MSIX 包
   - **iOS**：生成 Xcode 项目
5. 下载生成的安装包

#### 步骤 3：安装生成的包

**Android APK 安装**：
```bash
# 启用未知来源安装
设置 → 安全 → 未知来源 → 允许

# 安装 APK
adb install todo-list.apk
```

**Windows MSIX 安装**：
1. 双击 `.msix` 文件
2. 点击 **"安装"**
3. 应用将出现在开始菜单

---

### 方式三：本地部署（离线使用）

**特点**：无需网络，局域网内使用

#### 使用 Python 快速部署：

```bash
# 进入 dist 文件夹
cd dist

# Python 3
python -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080
```

然后在浏览器访问：`http://localhost:8080`

#### 使用 Node.js 部署：

```bash
# 安装 http-server
npm install -g http-server

# 进入 dist 文件夹
cd dist

# 启动服务
http-server -p 8080
```

#### 使用 Nginx 部署：

```nginx
server {
    listen 80;
    server_name localhost;
    
    location / {
        root /path/to/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
}
```

## 📝 使用说明

2. 输入任务内容
3. 选择优先级（低/中/高）

- 支持离线使用
- 清除浏览器数据将删除任务

## 🆘 常见问题

### Q: 如何在 Windows 上创建桌面快捷方式？

**A**: 安装 PWA 后，系统会自动创建快捷方式。如需手动创建：

1. 打开 Chrome/Edge
2. 访问应用网址
3. 点击地址栏右侧的安装图标
4. 安装后，右键桌面 → 新建 → 快捷方式
5. 输入：`chrome.exe --app=https://your-domain.com/`

### Q: 如何备份数据？

**A**: 数据存储在浏览器本地存储中：

```javascript
// 导出数据
const data = localStorage.getItem('daily-todo-app');
console.log(data);

// 导入数据
localStorage.setItem('daily-todo-app', data);
```

### Q: 如何生成真正的原生 APK？

**A**: 使用 Trusted Web Activity (TWA)：

1. 安装 Bubblewrap CLI：
   ```bash
   npm install -g @bubblewrap/cli
   ```

2. 初始化项目：
   ```bash
   bubblewrap init --manifest=https://your-domain.com/manifest.json
   ```

3. 构建 APK：
   ```bash
   bubblewrap build
   ```

### Q: 如何生成 Windows 可执行文件？

**A**: 使用 Electron 或 PWA Builder：

**方法 1 - Electron（需要安装 Node.js）**：

```bash
# 安装 Electron
npm install electron --save-dev

# 创建 main.js
# 修改 package.json
# 运行打包
npx electron-builder
```

**方法 2 - PWA Builder（推荐）**：

1. 访问 https://www.pwabuilder.com
2. 输入 PWA 网址
3. 选择 "Windows"
4. 下载生成的 MSIX 包
